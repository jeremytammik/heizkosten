# heizkosten &ndash; heruco4l

Housing unit heating and running cost allocation
[node.js](https://nodejs.org) app
and [mongodb](https://www.mongodb.com) database.

UIs required:

- Admin, see everything
- Consumption, meters
- Hausverwalter
- Nebenkosten (Vermieter)
